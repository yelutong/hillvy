1.安装nodejs  https://nodejs.org/en/
  在命令行中输入：node -v 和 npm -v 验证版本
2.进入node安装目录 安装cnmp    
   npm install -g cnpm Cregistry=https://registry.npm.taobao.org


    npm config get registry  //检查淘宝镜像是否安装成功
    或 npm info express



3.安装Angular Cli 
   之前电脑上如果安装过angular cli老版本的话建议先卸载，然后重新安装。

卸载老版本： 
npm uninstall -g @angular/cli  //卸载angular/cli

npm uninstall Csave-dev angular-cli 
卸载新版本： 
npm uninstall -g @angular/cli 
清除下缓存 
npm cache clean

然后开始安装，记住，要用cnpm 
cnpm install -g @angular/cli@latest 或者 npm install npm -g 要记住全局更新

若没安装过，直接cnpm install -g @angular/cli

  ng -v  //查看版本能否正常显示
4.创建项目
  ng new angular01  //angular01是项目名称
  进入所建目录启动服务  
   cd angular01
   cnpm install     //安装依赖 或者 npm install

   ng serve --open  //启动服务
   或者

   npm start 启动
